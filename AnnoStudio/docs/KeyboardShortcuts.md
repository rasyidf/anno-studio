# Anno Studio Keyboard Shortcuts Quick Reference

## General Commands

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+N` | New Canvas | Create a new canvas document |
| `Ctrl+O` | Open | Open an existing layout file |
| `Ctrl+S` | Save | Save current document |
| `Ctrl+Shift+S` | Save As | Save document with new name |
| `Ctrl+W` | Close | Close current document |
| `Ctrl+Q` | Exit | Exit application |

## Edit Commands

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo last action |
| `Ctrl+Y` | Redo | Redo last undone action |
| `Ctrl+Shift+Z` | Redo (Alt) | Alternative redo shortcut |
| `Ctrl+A` | Select All | Select all objects on canvas |
| `Ctrl+D` | Duplicate | Duplicate selected objects |
| `Delete` | Delete | Delete selected objects |
| `Backspace` | Delete (Alt) | Alternative delete shortcut |
| `Escape` | Clear Selection | Deselect all objects |

## Tool Selection

| Shortcut | Tool | Description |
|----------|------|-------------|
| `V` | Select Tool | Select and transform objects |
| `S` | Stamp Tool | Place building stamps |
| `R` | Rectangle Tool | Draw rectangular building areas |
| `L` | Line Tool | Draw buildings in a line |
| `P` | Pen Tool | Free-hand building placement |

## Object Manipulation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `↑` | Nudge Up | Move selection up by 1 grid unit |
| `↓` | Nudge Down | Move selection down by 1 grid unit |
| `←` | Nudge Left | Move selection left by 1 grid unit |
| `→` | Nudge Right | Move selection right by 1 grid unit |
| `Shift+↑` | Nudge Up (Fast) | Move selection up by 10 grid units |
| `Shift+↓` | Nudge Down (Fast) | Move selection down by 10 grid units |
| `Shift+←` | Nudge Left (Fast) | Move selection left by 10 grid units |
| `Shift+→` | Nudge Right (Fast) | Move selection right by 10 grid units |

## View & Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl++` | Zoom In | Increase zoom level |
| `Ctrl+-` | Zoom Out | Decrease zoom level |
| `Ctrl+0` | Zoom Reset | Reset zoom to 100% |
| `Ctrl+1` | Zoom to Fit | Zoom to fit all objects |
| `Ctrl+G` | Toggle Grid | Show/hide grid |

## Context Menu Actions

### Object Context Menu (Right-click on object)
- **Delete** - Remove the object
- **Duplicate** - Create a copy
- **Bring to Front** - Move to top layer
- **Send to Back** - Move to bottom layer
- **Properties...** - Show object properties

### Canvas Context Menu (Right-click on empty area)
- **Paste** - Paste from clipboard
- **Select All** - Select all objects

## Tips & Tricks

### Multi-Selection
- Hold `Shift` while clicking to add/remove objects from selection
- Drag with Select Tool to box-select multiple objects

### Precision Placement
- Hold `Shift` while nudging for faster movement
- Grid snapping is enabled by default for accurate placement

### Tool Usage
- **Stamp Tool**: Click to place; hover to preview
- **Rectangle Tool**: Click and drag to define area; toggle fill/outline
- **Line Tool**: Click and drag to draw straight line of buildings
- **Pen Tool**: Click and drag for free-hand placement

### Workflow Tips
1. Use `V` to quickly switch to Select tool for adjustments
2. Press `Ctrl+D` to quickly duplicate selected objects
3. Use arrow keys for fine-tuning object positions
4. Right-click for quick access to common operations
5. Use `Escape` to deselect and start fresh

## Customization (Future)
Keyboard shortcuts will be customizable in Preferences > Keyboard Shortcuts
